# CØRE PDV — Sistema Operacional Comercial Híbrido 🔘

![Status](https://img.shields.io/badge/Status-Est%C3%A1vel-green.svg?style=for-the-badge)

O **CØRE PDV** é um sistema de ponto de venda (PDV) híbrido, desenvolvido para oferecer máxima performance e estabilidade em operações de balcão. Com uma arquitetura *offline-first*, o sistema garante que sua operação nunca pare, mesmo sem conexão com a internet.

---

## 🚀 Principais Funcionalidades

*   **⚡ Checkout de Alta Velocidade:** Interface otimizada para uso exclusivo via teclado, suporte a leitores de código de barras e atalhos rápidos.
*   **🌐 Híbrido & Offline:** Sincronização inteligente de dados. Trabalhe offline e deixe o sistema sincronizar em background.
*   **📦 Gestão de Estoque:** Controle rigoroso de entrada/saída, histórico de movimentações e alertas de estoque baixo.
*   **💰 Controle Financeiro:** Sangrias, suprimentos, fechamento de caixa e relatórios diários (Z).
*   **🤝 Gestão de Clientes:** Cadastro completo e controle de vendas a prazo (**Fiado**) com sistema de quitação.
*   **🛡️ Segurança Industrial:** Autenticação via BCrypt, Banco de Dados ACID (SQLite) e backups automáticos.

---

## 📦 Instalação e Uso

### Para Usuários (Download Direto)
Se você deseja apenas utilizar o sistema, baixe a versão mais recente em:
👉 **[Download CØRE PDV v1.0.0](https://github.com/Dibritto/CORE/releases/latest)**

1.  Baixe o arquivo `.exe`.
2.  Execute o instalador.
3.  Nas opções do Windows SmartScreen, clique em **"Mais informações"** e depois em **"Executar assim mesmo"**.

### Para Desenvolvedores (Rodando localmente)
1. Clone o repositório:
   ```bash
   git clone https://github.com/Dibritto/CORE.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie a aplicação:
   ```bash
   npm start
   ```
